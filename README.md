# 🚀 Lotshaus.dev

Vítej v mém osobním portfoliu. Jsem Petr, patnáctiletý český tvůrce a nadšenec do technologií. Rád stavím hardwarové projekty, provozuji domácí server (homelab) a navrhuji vlastní desky plošných spojů.

## 🛠️ O projektu

Tento repozitář obsahuje zdrojový kód pro můj osobní web a portfolio, které najdeš na adrese: [lotshaus.github.io](https://lotshaus.github.io).

## 🚀 Co stavím a používám

* **ESP32 Sentry Turret** – Autonomní věžička s počítačovým viděním (OpenCV) na Raspberry Pi.
* **Homelab Setup** – Vlastní infrastruktura na HP t640 s Proxmox VE.
* **3D Tisk na zakázku** – Tisk na tiskárně Bambu Lab P1S pro zákazníky z Česka.
